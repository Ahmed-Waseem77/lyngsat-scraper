# Sat-ire Database lyngsat Scraping Script

### Dependencies 

puppeteer 
fs

### Setup & Usage 

First Run
```
npm install 
```
Then Run 

```
npm start -- --help
```
To get started 

---
#### Available Arguements 

``npm start -- -v`` 
verbose output on terminal

``npm start -- -wl`` 
do not scrape satellte launches 

``npm start -- -wc`` 
do not scrape tv channels

``npm start -- -hs`` 
run in headless mode
